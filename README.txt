Snippet storage for Kbexpander. 
== Description ==

For creation and edit of snipets directly from Kbexpander you'll need the "Application Passwords" WP plugin. https://wordpress.org/plugins/application-passwords/

Features: 
* creates a custom post type and taxonomy. This way all the users with permissins to edit this CPT will be able to contribute to a shared database of snippets that will be displayed in Kbexpander.
* generates reports of snippet usage to 

== Installation ==

How to install the plugin and get it working.

1. Upload `kbexpander` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where do I get the application password? =

After installing https://wordpress.org/plugins/application-passwords/ you can generate a new application password from Users -> Your Profile
